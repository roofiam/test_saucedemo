# SauceDemo UI Test Automation

Demo pet project for automated UI testing of [SauceDemo](https://www.saucedemo.com/), a sample e-commerce application used for QA practice.

The project is built with Python, Pytest, Selenium WebDriver, and the Page Object Model pattern. It covers common user flows such as authentication, catalog browsing, product sorting, cart badge behavior, sidebar navigation, product details, and external social links.

## Tech Stack

- Python 3.12
- Pytest
- Selenium WebDriver
- WebDriver Manager
- python-dotenv
- Allure Pytest
- Ruff
- uv
- Docker and Docker Compose
